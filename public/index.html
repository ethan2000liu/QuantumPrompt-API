QuantumPrompt API Documentation

Enhance your prompts for AI chatbots like ChatGPT and Google Gemini

Authentication

All API endpoints require authentication using JWT tokens. Include the token in the Authorization header:

Authorization: Bearer YOUR_JWT_TOKEN

POST /api/auth/register

Register a new user account.

curl -X POST https://quantum-prompt-api.vercel.app/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "your_password"
  }'

POST /api/auth/login

Login to get JWT token.

curl -X POST https://quantum-prompt-api.vercel.app/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "your_password"
  }'

GET /api/auth/session

Get current user session and settings.

curl -X GET https://quantum-prompt-api.vercel.app/api/auth/session \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

POST /api/auth/logout

Logout and invalidate current session.

curl -X POST https://quantum-prompt-api.vercel.app/api/auth/logout \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

API Key Management

POST /api/keys

Add a new API key.

curl -X POST https://quantum-prompt-api.vercel.app/api/keys \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "gemini",
    "apiKey": "your-gemini-api-key"
  }'

GET /api/keys

List all API keys.

curl -X GET https://quantum-prompt-api.vercel.app/api/keys \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

DELETE /api/keys/:id

Delete an API key.

curl -X DELETE https://quantum-prompt-api.vercel.app/api/keys/KEY_ID \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

User Settings

GET /api/settings

Get user settings.

curl -X GET https://quantum-prompt-api.vercel.app/api/settings \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

PUT /api/settings

Update user settings.

curl -X PUT https://quantum-prompt-api.vercel.app/api/settings \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "useOwnApi": true,
    "selectedKeyId": "your-selected-key-id"
  }'

Prompt Enhancement

POST /api/prompt/enhance

Enhance a prompt using AI.

curl -X POST https://quantum-prompt-api.vercel.app/api/prompt/enhance \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Your prompt text here",
    "useOwnApi": true
  }'

Usage Logs

GET /api/usage

Get usage logs.

curl -X GET https://quantum-prompt-api.vercel.app/api/usage \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
Note: Replace YOUR_JWT_TOKEN with the actual JWT token received after login. Replace KEY_ID with the actual UUID of the API key you want to delete. Replace your-selected-key-id with the actual UUID of the API key you want to select.
